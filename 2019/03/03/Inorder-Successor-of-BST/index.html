    Inorder Succussor in BST

    题目：

    Given a binary search tree and a node in it, find the in-order successor of that node in the BST.

    If the given node has no in-order successor in the tree, return null.

    中文题意：给一颗树的根节点和树中的一个结点，问按照中序遍历这个结点的下一个结点是谁

    解题思路一：

    1. 如果这个结点的右子树不为空，那么他的下一个结点就应该是右子树中最左下的那个结点；

    2. 但是如果这个结点没有右子树，那么他的下一个结点按照中序遍历应该是他的最近的那个祖先。也就是说碰到这种情况，我们要维护每个结点的祖先。

    3. 怎么维护他的祖先？那应该按照在中序遍历到这个给的结点之前应该不断地将遍历到的结点的祖先用一个暂时的变量记录下来。

    4. 怎么找到p结点的最近祖先结点？？由于这是一颗二叉搜索树，我们在寻找p结点是可以通过p结点的值和当前结点的值作比较，如果p结点的值大于当前结点的值，那么根据二叉搜索树定义，p结点在当前结点的右子树中。否则p结点在当前结点的左子树中，此时我们应该记录当前结点为最近祖先，这是因为中序遍历的顺序为左根右，所以在这个时候才是p结点的最近祖先。这样循环操作后找到p时结束。如果p结点有右子树，那么我们通过找到右子树中最左下角的结点就是中序遍历下p结点的下一个结点，如果没有右子树，那么下一个结点则为刚刚维护的最近祖先结点
